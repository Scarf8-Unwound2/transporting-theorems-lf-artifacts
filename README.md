# Artifacts for Transporting Theorems about Typeability in LF Across Schematically Defined Contexts

This repository contains the artifacts for the paper "Transporting Theorems
about Typeability in LF Across Schematically Defined Contexts." This includes
Adelfa proofs and encoded signatures for the examples in the paper. In order to
run these files, you will need to have the Adelfa proof assistant installed. You
can find instructions for installing Adelfa [on its
webpage](https://adelfa-prover.org/).

- Figure 2 is translated into an Adelfa signature in the file
  [`equal-untyped.lf`](/equal-untyped.lf).
- Example 1 is the `refl` theorem in the file
  [`equal-untyped.ath` on line 7](https://github.com/Scarf8-Unwound2/transporting-theorems-lf-artifacts/blob/main/equal-untyped.ath#L7).
- Section 6's property is stated as the `aeq-complete` theorem in the file
  [`equal-untyped.ath` on line 26](https://github.com/Scarf8-Unwound2/transporting-theorems-lf-artifacts/blob/main/equal-untyped.ath#L26).
